# Aurelia Beginner Kit ES2016
Setting up Aurelia Beginner Kit in the Cloud9 workspace

Based on the [Aurelia Documentation](http://aurelia.io/docs.html#/aurelia/framework/1.0.0-beta.1.2.2/doc/article/getting-started)

Install Dependencies
```
npm install
```

Install http-server
```
npm install -g http-server
```

Run Server
```
http-server
```

Open in Browser
```
http://{workspace-name}-{username}.c9users.io:8080
```
